#### 功能
##### 1.客戶端：
<br>分成兩種權限：一般客戶(3)、鎖帳客戶(4)
<br>(1)可進行存提款與轉帳
<br>(2)可查看與印出個人交易明細
<br>(3)修改個人資料
##### 2.後台端：
<br>分成兩種權限：一般人員(2)、管理者(1)
<br>(1)開戶
<br>(2)帳戶資料修改：僅後台可修改客戶權限、姓名、交易碼
<br>(3)可查看所有使用者資料與交易明細
<br>(4)刪除帳戶：此功能僅管理者可執行
