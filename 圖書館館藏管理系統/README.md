# 愛讀公共圖書館（Library Management System）

Java 8 + Swing（WindowBuilder 友善）+ Maven + MySQL 的圖書館管理系統示範專案。  
分為 **讀者端** 與 **館員後台** 兩種角色，涵蓋借閱/預約/歸還審核、個人信箱、讀書心得核可與公開牆、Excel 報表等功能。

---

## 1. 專案特色

- **雙角色登入**：讀者 / 館員（後台）
- **讀者端主頁**
  - 最新館藏列表（**每頁 10 筆**，可翻頁）
  - 關鍵字搜尋（書名/作者/ISBN/分類）
  - 會員中心、個人信箱（未讀提醒）、讀書心得、公開心得牆入口
- **借閱/預約流程**
  - 書籍狀態（books.state）
    - `0` 可借閱：只顯示「借閱」
    - `1` 借出中：只顯示「預約」
    - `2` 歸還待審核：只顯示「預約」
    - `3` 毀損/遺失：**讀者端不顯示/不可操作**（後台仍可查）
  - 讀者歸還採「**送出申請 → 館員核可**」模式
- **館員後台**
  - 書籍匯入（Excel 批次 / 單筆手動）
  - 書籍資料查詢（全部 / 非閒置中）
  - 讀者資料（註冊/修改/查詢）
  - 歸還待核可清單（核可後更新書況與備註）
  - 讀書心得核可/發佈（核可可發點數、可決定是否公開）
  - Excel 報表輸出
- **UI 主題統一**（UITheme）＋ **Icon Button 版面**（圖示按鈕固定大小、淺底，黑色線稿圖示清楚可見）
- **中文字顯示相容**：UI 字型改為 `SansSerif`（跨平台較穩定）

## 2. 環境需求

- JDK 8
- Maven 3.x
- MySQL 8.x（建議 8.0+）
- IDE：Eclipse（推薦）/ IntelliJ IDEA 皆可

---

## 3. 資料庫初始化

1) 建立資料庫與資料表（含示範資料、VIEW）
- 專案根目錄提供：`demo_library_db.sql`
- 在 MySQL 執行：

```sql
SOURCE demo_library_db.sql;
```

2) 設定資料庫連線（application.properties）

檔案位置：`src/main/resources/application.properties`

```properties
db.url=jdbc:mysql://localhost:3306/library_db?useSSL=false&serverTimezone=Asia/Taipei&useUnicode=true&characterEncoding=utf8
db.user=root
db.password=1234
```

> 連線字串已包含 `useUnicode=true&characterEncoding=utf8`，確保中文欄位讀取/顯示穩定。

## 4. 如何執行

### 方式 A：Eclipse（推薦）
1. Import Existing Maven Project  
2. 確認 Maven dependencies 都已下載
3. 執行：`src/main/java/controller/LoginUI.java`

### 方式 B：命令列
```bash
mvn clean package
```
再用 IDE 執行 `controller.LoginUI`

## 6. 主要功能操作說明（給使用者）

### 6.1 讀者端
1. **登入（讀者）**
2. **主頁**
   - 看到「館藏列表」與「搜尋框」
   - 點選表格某一本書，搜尋框會自動帶入該書名
3. **借閱/預約**
   - 點「放大鏡」進入借閱/預約頁
   - 系統依書籍狀態控制按鈕：
     - `0`：可借閱
     - `1`/`2`：可預約
4. **歸還**
   - 在借閱紀錄中對已借出書籍送出歸還申請
   - 送出後書籍狀態會進入 `2 歸還待審核`
   - 等館員核可後，若完好會恢復 `0 可借閱`，若毀損/遺失則進入 `3`
5. **個人信箱**
   - 有未讀訊息時，主頁會顯示紅色驚嘆號提醒
6. **讀書心得**
   - 可撰寫心得並送出
   - 需館員核可後可選擇是否在公開牆顯示
7. **公開心得牆**
   - 顯示已核可且設為公開的心得

### 6.2 館員後台
1. **登入（館員）**
2. **匯入書籍**
   - 批次上傳：Excel 匯入（可下載範本）
   - 手動輸入：單筆新增
3. **書籍資料**
   - 查詢全部 / 查詢非閒置中（借出、待審核、毀損遺失）
4. **書籍歸還（待核可）**
   - 讀者送出歸還申請後會出現在清單
   - 館員核可時可選：完好 / 毀損 / 遺失
5. **讀書心得（核可/發佈）**
   - 待核可清單（state=0）
   - 勾選核可可發點數；勾選發佈則公開牆可見
6. **報表印出（Excel）**
   - 依畫面提示輸出報表
